Step 3 - Create Models

You must choose a set of training features that you think will inform the predictions. Distil will automatically generate models to predict the target using the training features.

Training Set Selection View Summary

Selecting the Training Features

To choose the training features that your model should use in its predictions:

  1. Click Add to Training Set under any feature. In this example, add horsepower and cylinders.

    Distil adds the selected features to the list of Training Features. You can filter or change the typeÂ  of any the training features.

  2. Note the distribution of categories for the cylinders feature. Four, six and eight cylinder engines appear often in the dataset, while three and five cylinder engines appear much less frequently. Click the infrequent categories to omit them from the model.

    Greyed out categories are not included in the model.

  3. Review the updated sample table. Click the column headers to sort by feature and check for extreme values that may indicate problems with the data. Note that one row has a value of 0 for horsepower, which is likely an error.

  4. To omit records with this value, drag the slider at the beginning of the distribution of horsepower values to the right so the range starts above 0.

To remove a feature from the training set:
  • Click Remove under the feature name in the Training Features pane.

Generating the Models

Once you have a target feature and a set of training features, you can begin model generation. Based on your selections, Distil:

  1. Automatically determines the number and type of models to predict your target feature.
  2. Uses a subset of the actual data to train the models to predict the target from the training features.
  3. Applies the trained models to the rest of the actual data to predict the target feature value in each record.
Important: You must select a target feature and at least one training feature before you can generate the models.
To begin the model generation with the current feature selections:
  1. Click Create Models.
  2. Choose the maximum number of models that Distil should produce.

In this example, Distil will attempt to predict the acceleration based on the horsepower and cylinders features for a subset of the dataset.
