Step 3 - Create Models

You must choose a set of features that you think will inform the predictions. Distil will automatically generate models to predict the target using the features you select in this step.

Create Models View Summary

Selecting the Features to Model

To choose the features that your model should use in its predictions:

  1. Click Add under any feature. In this example, add horsepower and cylinders.

    Distil adds the selected features to the list of Features to Model. You can filter or change the typeÂ  of any of the features to model.

    To remove a feature from the features to model

    Click Remove under the feature name in the Features to Model pane.

  2. Note the distribution of categories for the cylinders feature. Four, six and eight cylinder engines appear often in the dataset, while three and five cylinder engines appear much less frequently. Click the infrequent categories to omit them from the model.

    Greyed out categories are not included in the model.

  3. Review the updated sample table. Click the column headers to sort by feature and check for extreme values that may indicate problems with the data. Note that one row has a value of 0 for horsepower, which is likely an error.

  4. To omit records with this value, drag the slider at the beginning of the distribution of horsepower values to the right so the range starts above 0.

Depending on your task, next you can:

  • Export the Problem to export the model definition without generating models or predicting acceleration
  • Generate the Models to predict acceleration using horsepower and cylinders
Important: You must select a target feature and at least one feature to model from before you can export the problem or generate the models.

Exporting the Problem

If your task is to discover a problem without generating a predictive model, you can now export your model definition. Otherwise, skip ahead to Generating the Models.

To export the problem:
  • Click Export Problem.

Generating the Models

If your task to build a predictive model, you can now begin the model generation. Based on your selections, Distil:

  1. Automatically determines the number and type of models to predict your target feature.
  2. Uses a subset of the actual data to train the models to predict the target using the features selected in this step.
  3. Applies the trained models to the rest of the actual data to predict the target feature value in each record.
To begin the model generation with the current feature selections:
  • Click Create Models.

In this example, Distil will attempt to predict the acceleration based on the horsepower and cylinders features for a subset of the dataset.

Next Steps

Review the generated models to determine whether they accurately predict the target feature.
